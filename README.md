## Cài đặt

```bash
cd AutoClickTool
pip install -r requirements.txt
```

## Chạy

```bash
python main.py
```

## Phím tắt

| Phím | Chức năng |
|------|-----------|
| F6 | Bắt đầu/Dừng Auto Click |
| F7 | Bắt đầu/Dừng Ghi thao tác |
| F8 | Phát lại thao tác đã ghi |
| F9 | Bắt đầu/Dừng Tìm hình ảnh |
| ESC | Dừng tất cả |
